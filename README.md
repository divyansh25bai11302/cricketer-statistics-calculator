# cricketer-statistics-calculator
Project Name - Cricketer Stats Calculator, Created By - Divyansh Mishra, Institute - VIT Bhopal

1. What the project does?
   
As the name suggests, the project gives the batting average and strike rate of a cricketer.

2. Why is the project useful?

The project is useful because it helps the user to easily calculate the batting average and strike rate of a cricketer by just entering total runs, balls played and no. of times player dismissed. It would help the scorer or data analyst to perform calculations easily.

3. How can the users get started with the project?

To get started, user must run the code. As the code runs, it will ask for 3 details, which are total runs, balls played and no. of times player dismissed. Enter the correct data and run it to get the stats correctly.
